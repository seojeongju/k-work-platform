WOW-CAMPUS 쿠키 사용 정책

시행일자: 2024년 1월 1일

WOW-CAMPUS(이하 "회사")는 웹사이트 및 서비스 운영을 위해 쿠키(Cookie)를 사용합니다. 이 정책은 회사가 어떤 쿠키를 사용하는지, 왜 사용하는지, 그리고 사용자가 쿠키를 어떻게 관리할 수 있는지에 대해 설명합니다.

쿠키란? 쿠키는 사용자가 웹사이트를 방문할 때 웹브라우저에 저장되는 작은 데이터 파일입니다. 쿠키는 사용자의 웹사이트 경험을 개선하고 개인화된 서비스를 제공하는 데 도움이 됩니다.

1 쿠키 사용 목적

필수 쿠키

  • 로그인 상태 유지
  • 보안 기능 제공
  • 사이트 기본 기능 운영
  • 언어 및 지역 설정

성능 쿠키

  • 웹사이트 사용 통계
  • 페이지 로딩 속도 개선
  • 사용자 경험 최적화
  • 오류 모니터링

기능 쿠키

  • 개인화된 콘텐츠 제공
  • 채팅 기능 지원
  • 사용자 선호사항 저장
  • 소셜 미디어 연동

광고 쿠키

  • 맞춤형 광고 제공
  • 광고 효과 측정
  • 리타겟팅 광고
  • 광고 빈도 조절

2 사용하는 쿠키의 종류

쿠키명 목적 유형 만료기간 제공업체
JSESSIONID 세션 유지 필수 세션 WOW-CAMPUS
remember_token 로그인 상태 유지 필수 14일 WOW-CAMPUS
user_preferences 사용자 설정 저장 기능 1년 WOW-CAMPUS
_ga 웹사이트 분석 분석 2년 Google
_gid 사용자 구분 분석 24시간 Google
_fbp Facebook 픽셀 광고 3개월 Facebook

3 제3자 쿠키

회사는 서비스 개선 및 사용자 경험 향상을 위해 다음과 같은 제3자 서비스를 사용하며, 이들은 자체 쿠키를 설정할 수 있습니다:

Google Analytics

  • 목적: 웹사이트 트래픽 분석
  • 수집 정보: 페이지 조회수, 사용자 행동
  • 쿠키: _ga, _gid, _gat
  • 정책: Google 개인정보처리방침

Facebook Pixel

  • 목적: 광고 효과 측정
  • 수집 정보: 전환 추적, 사용자 행동
  • 쿠키: _fbp, _fbc
  • 정책: Facebook 데이터 정책

Hotjar

  • 목적: 사용자 경험 분석
  • 수집 정보: 히트맵, 세션 녹화
  • 쿠키: _hjid, _hjFirstSeen
  • 정책: Hotjar 개인정보정책

채팅 서비스

  • 목적: 고객 상담 지원
  • 수집 정보: 채팅 세션, 사용자 ID
  • 쿠키: chat_session, visitor_id
  • 보존기간: 30일

4 쿠키 관리 방법

쿠키 설정 관리

사용자는 쿠키 배너 또는 설정 페이지를 통해 쿠키 사용에 대한 동의를 관리할 수 있습니다.

브라우저별 쿠키 관리 방법

Google Chrome

  1. 1. 설정 → 고급 → 개인정보 및 보안
  2. 2. 사이트 설정 → 쿠키 및 사이트 데이터
  3. 3. 모든 쿠키 및 사이트 데이터 보기
  4. 4. 특정 사이트 검색 후 삭제

Mozilla Firefox

  1. 1. 옵션 → 개인정보 및 보안
  2. 2. 쿠키 및 사이트 데이터
  3. 3. 데이터 관리
  4. 4. 해당 사이트 선택 후 제거

Safari

  1. 1. 환경설정 → 개인정보 보호
  2. 2. 웹사이트 데이터 관리
  3. 3. 특정 웹사이트 선택
  4. 4. 제거 버튼 클릭

Microsoft Edge

  1. 1. 설정 → 사이트 권한
  2. 2. 쿠키 및 사이트 데이터
  3. 3. 모든 쿠키 및 사이트 데이터 보기
  4. 4. 사이트별로 관리 및 삭제

모바일 기기 쿠키 관리

iOS (iPhone/iPad)
  • • 설정 → Safari → 고급 → 웹사이트 데이터
  • • 모든 웹사이트 데이터 제거 또는 개별 제거
Android
  • • Chrome 앱 → 더보기 → 설정 → 개인정보 보호
  • • 인터넷 사용 기록 삭제 → 쿠키, 사이트 데이터

5 쿠키 거부 시 영향

쿠키 사용을 거부하거나 삭제하면 다음과 같은 제한이 있을 수 있습니다:

제한사항

  • 자동 로그인 불가
  • 개인화 서비스 제한
  • 설정 정보 저장 불가
  • 일부 기능 이용 제한

가능한 기능

  • 기본 웹사이트 이용
  • 공개 콘텐츠 조회
  • 수동 로그인
  • 검색 기능 이용

6 개인정보보호

회사는 쿠키를 통해 수집된 정보의 보안과 프라이버시를 보호하기 위해 다음과 같은 조치를 취합니다:

암호화

민감한 쿠키 데이터는 암호화하여 저장

만료 관리

적절한 만료 시간 설정으로 불필요한 데이터 저장 방지

접근 제한

쿠키에 대한 접근 권한을 최소화
